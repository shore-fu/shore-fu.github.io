# Shore Fu 个人展示网站

欢迎来到我的 GitHub Pages 个人网站！🎉


## 功能特性

✨ **现代化设计** - 响应式布局，适配所有设备
🎯 **简洁导航** - 流畅的页面导航体验
📱 **移动友好** - 完全支持移动设备
🎨 **美观样式** - 现代化的配色方案


## 网站结构

- **首页 (Hero)** - 欢迎区域和个人介绍
- **关于** - 个人简介和技能展示
- **项目** - 展示你的项目作品
- **联系** - 与访客互动的联系方式

## 文件说明

- `index.html` - 主页面结构
- `styles.css` - 网站样式表
- `script.js` - 交互脚本
- `README.md` - 本文件

## 快速开始

### 1. 启用 GitHub Pages
1. 前往仓库设置 (Settings)
2. 找到 Pages 部分
3. 选择源分支为 `main`
4. 保存

### 2. 自定义内容

编辑 `index.html` 来修改：
- 网站标题和描述
- 个人信息
- 社交媒体链接
- 项目内容

### 3. 自定义样式

编辑 `styles.css` 来修改：
- 颜色主题 (在 `:root` 变量中)
- 字体大小
- 布局和间距
- 响应式设计

## 部署

此网站使用 GitHub Pages 自动部署。只需将更改推送到 `main` 分支即可。

网站将在以下地址可用：
```
https://shore-fu.github.io
```
